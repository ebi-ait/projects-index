Aims

The Human Cell Atlas (HCA) Project Catalogue by EMBL-EBI aims to provide a comprehensive list of all the cellular resolution data generated by and available to the HCA community.

By providing links to the publications and the resources where the data is located, the catalogue allows quick access to all newly generated data in the HCA community, and accelerates collaboration in the building of the atlas.

The list is curated by the HCA Data Coordination Platform wrangling team at EMBL-EBI, but is open for contribution by anyone who works within the single cell technology field.

To send a suggestion for the Catalogue, use the Suggest a project form.

Eligibility Criteria

There are three main categories of data considered eligible for the Human Cell Atlas, and respectively, the HCA Project Catalogue.

Data directly contributed to the HCA Data Coordination Platform

To be eligible for this category, datasets must:

  • Be contributed to the Human Cell Atlas Data Coordination Platform by the data owner, with support of the HCA DCP data wranglers.
  • Have either a pre-print publication referencing data that meets the criteria for published data (see below), or a supporting justification as to why it should be included in the HCA.
  • Be available in raw fastq format and/or as a gene expression matrix (raw counts or processed counts). Data from multiple samples must be demultiplexed (each fastq file contains data from a single sample) to be considered eligible. If only multiplexed data is available, gene expression matrices will still be considered eligible.

Data from official Human Cell Atlas publications

To be eligible for this category, datasets must:

  • Support a publication that has been reviewed by the HCA publication committee and added to the official HCA publications list
  • Be available in raw fastq format and/or as a gene expression matrix (raw counts or processed counts). Data from multiple samples must be demultiplexed (each fastq file contains data from a single sample) to be considered eligible. If only multiplexed data is available, gene expression matrices will still be considered eligible.

Other published data

To be eligible for this category, datasets must:

  • Be referenced in a peer-reviewed journal or be available as pre-print
  • Contain at least some samples derived from healthy human donors
  • Be derived from a recognised single cell technology.
  • Be available in raw fastq format and/or as a gene expression matrix (raw counts or processed counts). Data from multiple samples must be demultiplexed (each fastq file contains data from a single sample) to be considered eligible. If only multiplexed data is available, gene expression matrices will still be considered eligible.

Prioritisation

  1. Data directly contributed to the HCA Data Coordination Platform
    • From co-funded collaborators (e.g. WSSS, MRC, HCA:Organoid, Seed Networks), then
    • First come first serve
  2. Data from official Human Cell Atlas publications
    • Newest first, oldest last
    • Published Data on primary tissues
    • Poorly represented donor sample locations first (i.e. US, European donors will be actively down-prioritised). Once data from all possible locations is captured, then
    • Organs with least coverage first, organs with most coverage last
  3. All other eligible published data scored by a combination of:
    • Organ coverage in the HCA, and
    • Cell count size, and
    • Commonality of technology

Data sources

HCA Project Catalogue contains projects which are selected and curated by the HCA Data Coordination Platform wranglers, from these main sources:

  • The official HCA publications list
  • The HCA Data Portal
  • Projects are funded to contribute data to the HCA including:
    • Wellcome Strategic Science Support Award
    • MRC Developmental Cell Atlas
    • Organoid Horizon 2020 award
    • CZI seed network funding
    • CZI biological network funding
  • Established data archives
    • GEO
    • ArrayExpress
  • Specialised databases
    • Single Cell Expression Atlas
    • The Single cell studies database, maintained by Valentine Svensson
    • The COVID19 Cell Atlas
  • Suggestions from the HCA community (that means you!)

Contribute a project

This catalogue is created for and by the HCA community and aims to provide quick and easy access to suitable data. Help us build a comprehensive catalogue by letting us know if we have missed a project.

The Data wrangling team can support the inclusion of these types of projects:

  • A project which is already published and the data is available elsewhere. The data must be available in raw fastq format and/or as a gene expression matrix (raw counts or processed counts). Data from multiple samples must be demultiplexed to be considered eligible. If only multiplexed data is available, gene expression matrices will still be considered eligible.
  • A project, which is in the process of publication, and data is pending or needs an accession
  • A project, which is under way, but there is no publication or data available yet. (The Data Wrangling team can support you in gathering and organising its metadata)
