Aims

The Human Cell Atlas (HCA) Project Catalogue by EMBL-EBI aims to provide a comprehensive list of all the cellular resolution data generated by and available to the HCA community.

By providing links to the publications and the resources where the data is located, the catalogue allows quick access to all newly generated data in the HCA community, and accelerates collaboration in the building of the atlas.

The list is curated by the HCA Data Wrangling team at EMBL-EBI, but is open for contribution of any suitable projects. To suggest a project for the catalogue which we have missed out, send an email to the data wrangler team .

Data sources

HCA Project Catalogue contains projects which are selected and curated by the HCA Data Coordination Platform wranglers, from these main sources:

  • The official HCA publications list
  • The HCA Data Portal
  • Projects a funded to contribute datato the HCA including:
    • Wellcome Strategic Science Support Award
    • MRC Developmental Cell Atlas
    • Organoid Horizon 2020 award
    • CZI seed network funding
    • CZI biological network funding
  • Established data archives
    • GEO
    • ArrayExpress
  • Specialised databases
    • Single Cell Expression Atlas
    • The Single cell studies database, maintained by Valentine Svensson
    • The COVID19 Cell Atlas
  • Suggestions from the HCA community (that means you!)

Contribute a project

This catalogue is created for and by the HCA community and aims to provide quick and easy access to suitable data. Help us build a comprehensive catalogue by letting us know if we have missed a project.

The Data wrangling team can support the inclusion of these types of projects:

  • A project, which is already published and the data is available elsewhere
  • A project, which is in the process of publication, and data is pending or needs an accession
  • A project, which is under way, but there is no publication or data available yet. (The Data Wrangling team can support you in gathering and organising its metadata)
